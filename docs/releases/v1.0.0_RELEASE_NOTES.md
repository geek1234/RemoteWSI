# RemoteWSI v1.0.0 Release Notes / 首个开源版本发布说明

Release Date: April 19, 2026  
Tag: `v1.0.0`

---

## 中文（论文成果风格）

### 版本概述
`v1.0.0` 为 RemoteWSI 首个开源版本，形成了“跨院传输 + 在线解码阅片 + 在线颜色标准化”的完整工程闭环，可作为学位论文的工程化成果支撑版本。

### 核心贡献
1. 构建了面向病理全视野图像（WSI）的端到端应用流程：医院A采集与传输、医院B导入与在线评阅。  
2. 支持 `.svs`、`.tif/.tiff`、`.jpg/.png` 等格式，并针对超大切片采用金字塔低倍率优先策略，提高首屏和交互响应速度。  
3. 集成在线 GDDN(A,B) 颜色标准化：用户可在界面选择源图像A和目标图像B，在线执行标准化并可视化结果。  
4. 提供多结果联动可视化：Source/Target/Normalized(Tiled)/Normalized(Mono)/Difference Heatmap，并支持结果卡片点击切换到主视图放大评阅。  
5. 完成评阅交互增强：信息区与 GDDN 区域均支持折叠与高度调节，便于临床或科研场景下的工作空间管理。  
6. 完成 Windows 封装与开源基线建设：提供可分发应用包、源码发布包、CI、贡献规范与安全披露文档。

### 工程与复现
- 本地运行入口：`app.local_main:app`、`app.local_launcher:main`  
- Windows 启动脚本：`scripts/run-local-viewer.ps1`  
- UI/接口回归脚本：`scripts/test-local-viewer.ps1`、`scripts/test-local-ui.ps1`、`scripts/test-color-normalization.ps1`、`scripts/test-color-normalization-ui.ps1`  
- Windows 封装脚本：`scripts/build-windows-bundle.ps1`  
- 源码发布脚本：`scripts/build-source-release.ps1`

### 发布资产
- Windows 应用包：`LocalWSIViewer-windows.zip`  
- 开源源码包：`wsi-viewer-main-source-20260419.zip`

### 学术价值说明
本版本体现了“算法—系统—场景”一体化落地：  
在算法层面实现病理图像颜色标准化的在线服务化；在系统层面实现高分辨率切片远程协作链路；在应用层面形成可复现实验与可部署软件成果，为多中心病理研究与远程评阅提供可扩展的技术基线。

### 已知限制与后续方向
- 当前标准化结果以在线预览与对比分析为主，超高倍率全分辨率标准化仍建议离线批处理。  
- 后续将继续完善多用户权限、审阅标注协同与更细粒度性能监控能力。

## English (Thesis-Oriented Style)

### Overview
`v1.0.0` is the first open-source release of RemoteWSI, delivering an end-to-end pipeline for **cross-site transfer, online slide decoding/review, and online color normalization**, suitable as an engineering outcome in a thesis context.

### Major Contributions
1. Established a full workflow for pathology WSI collaboration between institution A (acquisition/transfer) and institution B (import/review).  
2. Supported major pathology formats (`.svs`, `.tif/.tiff`, `.jpg/.png`) with pyramid-aware low-magnification preference for faster initial rendering on large slides.  
3. Integrated online GDDN(A,B) color normalization with direct in-app source/target slide selection.  
4. Delivered multi-view result visualization: Source, Target, Normalized (Tiled), Normalized (Mono), and Difference Heatmap.  
5. Added reviewer-centric interaction upgrades: click-to-promote result cards to the main viewport, plus collapsible/resizable Info and GDDN panels.  
6. Completed Windows packaging and open-source baseline setup, including CI and contribution/security documentation.

### Reproducibility & Tooling
- Local entrypoints: `app.local_main:app`, `app.local_launcher:main`  
- Windows launcher: `scripts/run-local-viewer.ps1`  
- Smoke/regression tests:  
  `scripts/test-local-viewer.ps1`  
  `scripts/test-local-ui.ps1`  
  `scripts/test-color-normalization.ps1`  
  `scripts/test-color-normalization-ui.ps1`  
- Packaging scripts:  
  `scripts/build-windows-bundle.ps1`  
  `scripts/build-source-release.ps1`

### Release Assets
- Windows binary bundle: `LocalWSIViewer-windows.zip`  
- Source release bundle: `wsi-viewer-main-source-20260419.zip`

### Research Significance
This release demonstrates a practical integration of **algorithm, system, and deployment scenario**: algorithmic color normalization is operationalized as an online service, and WSI remote review is delivered as a reproducible, deployable software artifact for multi-center pathology collaboration.

### Known Limitations & Future Work
- Current normalization output is optimized for online preview and comparative review; full-resolution ultra-high-magnification normalization is still better handled offline.  
- Future versions will focus on multi-user access control, collaborative annotations, and finer-grained observability/performance telemetry.
